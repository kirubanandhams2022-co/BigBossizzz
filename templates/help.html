Complete Host Guide

Step-by-Step Host Instructions

  1. Access Quiz Creation: Click "Create Quiz" in your dashboard
  2. Quiz Details:
    • Enter a descriptive title
    • Add clear instructions for participants
    • Set time limit (recommended: 1-3 minutes per question)
    • Choose proctoring security level
  3. Adding Questions:
    • Multiple Choice: Best for objective assessment
    • True/False: Quick evaluation questions
    • Text Answer: For detailed responses
  4. Proctoring Settings:
    • Enable camera monitoring
    • Set violation thresholds
    • Configure background noise detection
    • Enable mobile device blocking
  5. Publishing: Activate quiz when ready for participants

Participant Guide

How to Take a Monitored Quiz
Important: Use only desktop computers. Mobile devices are blocked for security.
Before Starting
  1. Verify Account: Check your email for account verification
  2. Login: Use provided credentials
  3. Test Equipment:
    • Ensure webcam is working
    • Test microphone permissions
    • Stable internet connection
    • Updated browser (Chrome/Firefox recommended)
During the Quiz
✅ Allowed Actions
  • Stay focused on quiz tab
  • Look at your screen
  • Take notes (if permitted)
  • Ask for help via chat
❌ Prohibited Actions
  • Switching browser tabs
  • Using mobile devices
  • Talking to others
  • Looking away from screen
  • Multiple people in frame

Troubleshooting

Common Issues & Solutions

  1. Grant camera permission when prompted
  2. Check if camera is used by another application
  3. Refresh the page and try again
  4. Use Chrome or Firefox browser
  5. Check privacy settings in browser

Solution: Use a desktop or laptop computer. Mobile devices and tablets are not permitted for security reasons.

If you're on desktop and still seeing this message:

  1. Clear browser cache and cookies
  2. Use a different browser
  3. Check if mobile emulation is enabled in browser

To prevent false alerts:

  • Take quiz in a quiet room
  • Turn off phones and notifications
  • Avoid talking during the exam
  • Use headphones if needed (with host permission)

System Requirements

Minimum Requirements
  • Device: Desktop or Laptop only
  • Internet: Stable broadband connection
  • Browser: Chrome 90+, Firefox 88+
  • Camera: Working webcam
  • Microphone: Built-in or external mic
  • JavaScript: Must be enabled
Not Supported
  • Mobile phones
  • Tablets and iPads
  • Internet Explorer
  • Safari (limited support)
  • Incognito/Private browsing
  • Virtual machines
Recommended: Chrome browser on Windows/Mac desktop with 8GB+ RAM for optimal performance.
